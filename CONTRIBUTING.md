## Quick Start

```bash
# Clone and setup
git clone https://github.com/zshyc/pdf-master.git
cd pdf-master
pip install -r requirements.txt
pip install pytest black flake8 mypy

# Run tests
pytest

# Format code
black skills/pdf/scripts/
```
