# PDF Report Output Style

当输出 PDF 处理结果时，使用以下结构化格式：

## 报告结构

```markdown
## 📄 PDF 处理报告

### 📊 基本信息
| 属性 | 值 |
|------|-----|
| 文件名 | ... |
| 页数 | ... |
| 大小 | ... |

### ✅ 操作结果
[操作详情]

### 📈 统计数据
[相关统计]

### ⚠️ 注意事项
[需要注意的问题]
```

## 输出原则

1. **结构化**：使用表格和列表组织信息
2. **可视化**：使用 emoji 标记不同类型的信息
3. **简洁**：突出关键信息，避免冗余
4. **可操作**：提供明确的下一步建议
