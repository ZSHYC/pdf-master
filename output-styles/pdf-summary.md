# PDF Summary Output Style

当输出 PDF 内容摘要时，使用以下格式：

## 摘要结构

```markdown
## 📝 PDF 摘要

### 核心内容
[一句话概括]

### 关键要点
1. ...
2. ...
3. ...

### 数据亮点
- 关键数字/统计
- 重要发现

### 建议行动
- [ ] 行动项 1
- [ ] 行动项 2
```

## 输出原则

1. **精炼**：控制在 500 字以内
2. **要点化**：使用列表而非段落
3. **可执行**：提供具体建议
4. **层次分明**：从核心到细节
